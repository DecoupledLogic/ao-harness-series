# Release v0.1.0

Slice: S0, disposable Microsoft Agent Framework spike.
Proves: Rung 0, touch the primitives before you theorize about them.

## What shipped

| ID | Title | Spec |
|----|-------|------|
| S0 | Throwaway Microsoft Agent Framework spike | [SPEC-S0](../../specs/SPEC-S0-maf-spike.md) |

A single-file, disposable spike at `spikes/maf-spike/` that exercises the Microsoft Agent Framework primitives (`IChatClient`, `ChatClientAgent`, `AgentSession`, `AIFunctionFactory`) end to end against a real provider. It lives outside the kernel solution and is marked throwaway: once Slice 1 lands, it can be deleted. The lesson it earns is rung 0, run the primitives before theorizing about them.
